# #キャクモクルー LP

飲食店専門SNS運用代行サービス「#キャクモクルー」のランディングページです。

## ファイル構成

```
LP/
├── index.html          # メインHTMLファイル
├── css/
│   └── styles.css      # スタイルシート
├── js/
│   └── main.js         # JavaScript
├── images/             # 画像格納フォルダ（必要に応じて追加）
└── README.md           # このファイル
```

## 使用方法

### LINE URLの設定

`index.html` 内の `YOUR_LINE_URL_HERE` を実際のLINE公式アカウントURLに置換してください。

```html
<!-- 検索して置換 -->
YOUR_LINE_URL_HERE → https://line.me/R/ti/p/@xxxxx
```

該当箇所（5箇所）：
1. ファーストビューのCTAボタン
2. 実績セクション後のCTAボタン
3. 料金プランセクション後のCTAボタン
4. 最終CTAセクションのボタン
5. フローティングCTAボタン

### ローカルでの確認

ブラウザで `index.html` を直接開くか、ローカルサーバーで確認できます。

```bash
# Python 3の場合
python -m http.server 8000

# その後ブラウザで http://localhost:8000 にアクセス
```

## 主な機能

- **レスポンシブデザイン**: モバイル・タブレット・デスクトップに対応
- **スムーススクロール**: アンカーリンクでの滑らかなスクロール
- **スクロールアニメーション**: フェードイン効果
- **フローティングCTA**: スクロール後に画面下部に固定表示
- **FAQアコーディオン**: 開閉式の質問セクション
